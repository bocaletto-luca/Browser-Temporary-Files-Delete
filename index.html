Browser Temporary Files Delete

Cross-platform Python GUI to scan and remove temporary/cache files from major web browsers.

Author: Luca Bocaletto

License: GPLv3  |  View LICENSE

Source: GitHub Repository

Overview

  • Collects system-independent paths for browser caches (Chrome, Firefox, Edge, Opera, Safari)
  • Scan & delete selected browsers’ temporary files via a friendly PyQt5 interface
  • Run in background as a Windows Service or systemd daemon
  • Auto-restart on crash and auto-start at system boot

Screenshots

Installation

# Clone repository
git clone https://github.com/bocaletto-luca/Browser-Temporary-Files-Delete.git
cd Browser-Temporary-Files-Delete

# Install dependencies
pip install -r requirements.txt

Usage

  1. Launch the GUI: python your_application.py
  2. Check the browsers you want to clean
  3. Click “Start Scan” to see total temp files
  4. Click “Delete Files” to purge selected caches
  5. Use “About” menu to view app info

System Requirements

  • Python 3.7 or newer
  • PyQt5
  • OS module (standard library)

Service Installation

Windows Service

  1. Install:
    2. python windows_service.py install
  2. Configure auto-restart:
    3. sc.exe failure PyServerReport reset=0 actions=restart/5000
  3. Start / Stop / Remove:
    4. python windows_service.py start
python windows_service.py stop
python windows_service.py remove

Linux (systemd)

  1. Copy files:
    2. sudo mkdir -p /opt/browser-clean
sudo cp your_application.py config.yaml /opt/browser-clean/
sudo cp linux_service.service /etc/systemd/system/browser-clean.service
  2. Enable & start:
    3. sudo systemctl daemon-reload
sudo systemctl enable browser-clean
sudo systemctl start browser-clean
  3. Check status / logs:
    4. sudo systemctl status browser-clean
journalctl -u browser-clean -f
